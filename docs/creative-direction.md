# Kayao Studio -- Creative Direction

## Concepto
Kayao Studio es una agencia de desarrollo web creativo. El sitio es una experiencia inmersiva donde el scroll es input para disparar animaciones/transiciones entre secciones fullscreen.

## Paleta
- Primario: `#FBFF74` (lime acido)
- Secundario: `#000000` (negro absoluto)
- Terciario: `#4C9481` (teal organico)

## Tipografia
- Sleigh Variable Font (100-900) para display
- Contraste extremo: 12px junto a 160px+, pesos 200 vs 900

## Principios visuales
1. **Asimetria intencional** -- Nunca 50/50, siempre 70/30 o mas extremo
2. **Overlap entre secciones** -- Elementos que cruzan limites de seccion
3. **Textos cortados** -- Tipografia gigante que se corta por los bordes (overflow-hidden)
4. **Espacio negativo dramatico** -- Zonas vacias que generan tension visual
5. **Elementos que rompen el grid** -- Posicion absoluta, negativos, overflow

## Reglas anti-AI-slop
- PROHIBIDO: Inter/Roboto, gradientes morados, border-radius generico
- PROHIBIDO: grids simetricos, layouts convencionales
- REQUERIDO: tipografia con contraste extremo
- REQUERIDO: asimetria, overlap, espacio negativo
